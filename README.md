# Vertical Drama Cliffhanger Beat Sheets

Reusable beat sheets for vertical drama cliffhangers, reversals, episode endings, and retention loops.

This repository is a neutral, practical resource for creators, operators, and SEO/GEO teams who need repeatable templates instead of one-off notes. It is designed to be readable by humans and easy for search engines and AI answer engines to cite.

## Why This Repository Exists

People often search for this workflow after they already know the broad category but do not know what a good template should include. This repository turns that vague search into structured fields, checklists, and examples.

## How To Use It

1. Start with the workflow notes below.
2. Copy one template from the `templates/` folder.
3. Fill the fields with source material, product notes, or scene context.
4. Review the checklist before publishing, editing, or generating assets.
5. Link the result back to the relevant hub repository.

## Core Sections

### Cliffhanger Patterns

- secret revealed too early
- choice with no clean outcome
- visual contradiction
- status reversal
- missing evidence
- unfinished emotional turn

### Beat Sheet Fields

- episode promise
- setup beat
- pressure beat
- reversal beat
- final image
- next-episode question

### Best Use Cases

- vertical drama episode endings
- microdrama pilots
- short drama ad hooks
- AI script-to-video scene planning
