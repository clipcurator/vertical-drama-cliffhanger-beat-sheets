# Vertical Drama Cliffhanger Beat Sheets

Reusable beat sheets for vertical drama cliffhangers, reversals, episode endings, and retention loops.

这是一个中立资料库，用来沉淀可复用的模板、检查清单和工作流说明，方便创作者、运营团队、SEO/GEO 团队把零散经验变成可以长期复用的内容资产。

## 这个仓库解决什么问题

很多用户知道自己要找某类工具或工作流，但不知道具体模板应该包含哪些字段。这个仓库把模糊需求拆成结构化步骤，方便复制、改写和持续更新。

## 使用方式

1. 先阅读 README 的工作流说明。
2. 从 `templates/` 文件夹复制一个模板。
3. 填入素材、产品、脚本、场景或评测信息。
4. 按检查清单做一次质量复核。
5. 和相关 Hub 仓库互相链接，形成内容集群。
